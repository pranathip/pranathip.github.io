The Hype Advisor - Mobile Application

UX DESIGN MOCKUP • CONTRACT PROJECT • JANUARY 2020



Background: The Hype Advisor is a hedgefund that aims to democratize access to investment in the streetwear market.

Objective: The objective of this project was to create an interactive mockup that shows the design and usability of a HA mobile app that allows users to monitor the changing features of their investment.


Phase 1: Design
Started by laying out usability params and capabilites, laid out screen archetypes, used Procreate to sketch initial wireframe. Progressed to InVision and Figma to create more solid wireframes and mockups to pass on to implementation.

Phase 2: Scraper
Due to the non-existence of a Sneaker API to get data from, turned to creating my own StockX sneaker scraper. Used Selenium and BeautifulSoup to scrape Jordan 1 Data (test subset of ~800 sneakers) and loaded the information into a JSON file.

Phase 3: iOS Engineering
Used Objective-C and Parse backend to implement the application. Involved creating custom Sneaker and User objects, User Authentication, and calls to News API to display sneaker news data. Visit this link for more info on technical implementation.

Key Application Design Features:

  • >User registration/sign-in
  • Account Overview: showes the user's current capital, profit, and return over a given time period along with relevant graphics
  • Investment Breakdown: interactive pie-chart graphic that displays investment percentages of each show purchased, details on number/margin of capital used on a specific type of shoe, etc.
  • Investment Details: when tapping on a specific sneaker shows a breakdown of the shoe's price fluctuation over a given period of time, details about size/number/price purchased at, and historical facts surrounding the shoe's creation
  • News: top and relevant sneaker stories (similar to how news is displayed on the Robinhood application interface)
