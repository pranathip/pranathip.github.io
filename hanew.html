The Hype Advisor
MOBILE APP DESIGN MOCKUP • CONTRACT • JANUARY 2020
BACKGROUND
The Hype Advisor is a hedgefund that aims to democratize access to investment in the streetwear market. The group accepts applications for investors, and uses the acquired capital to invest in fast-growing streetwear items like shoes, clothing, and more.
DESIGN APPLICATION
In order to keep their investors aware and updated on the investments they're making, The Hype Advisor team wanted to begin designing/implementing a mobile app to allow users to see this information. The application would have the ability to show a user their investment amount, allow them to add more funds, and view the breakdown of their current investment/information about the items that are being invested in.
PROJECT OBJECTIVE
Create an interactive mockup that shows the design and usability of The Hype Advisor mobile app, using insights from user research.


—
PART 1
User Research
User Personas

The Hype Advisor has around 20-30 investors, and all of them have very uniform/similar knowledge on the streetwear industry. As a first step, I interviewed several investors, developing an archetypical user persona for the application with certain goals and requirements that could be met with the mobile application.

Screen Archetypes

After taking into account what the typical investor wanted in an application, I created a list of screens and functionalities that I wanted to prototype. Creating archetypes for each screen with detailed information on which screen transitions to which buttons, functions, etc. helps to create the final interactive wireframe without much trouble.

—
PART 2
